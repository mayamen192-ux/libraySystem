2.At least 3 examples of basic SELECT queries with explanations?
example 1: SELECT first_name, last_name, department 
FROM employees 
WHERE department = 'Sales' ORDER BY last_name;
This query retrieves the first and last names of employees 
in the Sales department, sorted alphabetically by last name.
example 2: SELECT * 
FROM employees;
Retrieves all columns and rows from the employees table.
example 3:SELECT first_name, last_name, department
 FROM employees;
Returns only the first name, last name, and department of each employee.

3.Examples of WHERE, ORDER BY, and DISTINCT usage?
example of where:SELECT first_name, last_name, department
